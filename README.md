# AutoInsight
AutoInsight is a plateform which provides details about automobile manufacturers, the models they build, and the type of vehicles they make. Application uses official APIs of [National Highway Traffic Safety Administration](https://www.nhtsa.gov/about-nhtsa) to provide accurate information.

## Prerequisite:
Below installations are required to run the project locally.

- npm: 
```sh
  npm install npm@latest -g
  ```
- Nodejs: https://nodejs.org/en
- Angular CLI:
```sh
  npm install -g @angular/cli
  ```
- Git: https://git-scm.com/downloads

## Steps to add project in local system:
1. Create a project folder on your system
2. Git clone repo - CMD --> Go to Project folder path --> 

```sh
  git clone https://github.com/malvi23/siemens-coding-exercise.git
  ```
3. In cmd, go to <Project_Folder> and Install client npm dependencies
```sh
  npm install
  ```
4. To start angular project, go to <Project_Folder>\photo-sharing-app-client in cmd and run:
```sh
  ng serve
  ```
5. Open browser and hit 'http://localhost:4200'
   
## Technology Stack
This application was built using [Angular v15.x] and uses the following technologies:

- RxJS: To handle async tasks
- Lodash: Data manipulation
- @angular/cdk: Virtual scroll  
- Jasmine, Karma: Unit testing
